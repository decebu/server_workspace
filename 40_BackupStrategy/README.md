# Backup-Strategie

Dieses Verzeichnis enthält die Backup-Dokumentation für das Homelab.

## Inhalt
- `clients/` – Backup-Profile und Konfigurationen für Clients
- `docs/` – Anleitungen für QNAP und Windows Backup
- `monitoring/` – Überwachung und Prüfung von Backup-Prozessen

## Verbindung zur DMS-Strategie
`paperless-ngx` muss als eigener offener Punkt in der Homelab-Backup-Strategie betrachtet werden.
Siehe hierzu auch:
- `../5_Dokumentenmanagement/40_Backup-Verweis.md`

> **Offen:** Detaillierte Backup-Strategie für `paperless-ngx` wird im Homelab-Konzept behandelt.
